# Eleve Chat Desktop

本地 AI 智能助手桌面客户端，内置前后端，一键安装即用。

## 安装

1. 双击 `Eleve Chat_0.1.0_x64-setup.exe` 运行安装程序
2. ⚠️ **Windows 安全提示处理**：由于应用未购买代码签名证书，Windows 可能弹出以下警告，按以下步骤操作即可正常安装：

   **SmartScreen 弹窗**（"Windows 已保护你的电脑"）：
   - 点击「更多信息」→ 点击「仍要运行」

   **Defender / 杀毒软件报毒**（误报）：
   - 这是 NSIS 安装包 + 未签名 exe 的常见误报
   - 在杀毒软件中选择「允许」或「恢复」该文件
   - 或临时关闭实时保护，安装完成后重新开启

3. 按提示完成安装，启动后应用会自动运行后端服务

## 卸载

- 控制面板 → 程序与功能 → 右键卸载
- 卸载会自动清理后端进程，无需手动操作

## 常见问题

**Q: 为什么杀毒软件报毒？**
A: 应用未购买代码签名证书，Windows 和杀软会对未签名的程序提高警惕。这是误报，程序完全安全。可向 Microsoft Defender [提交误报](https://www.microsoft.com/en-us/wdsi/filesubmission) 加速白名单。

**Q: 启动后白屏？**
A: 等待 5-10 秒，后端服务启动需要时间。若持续白屏，检查是否有其他程序占用了后端端口。

**Q: 如何更新？**
A: 下载新版本安装包覆盖安装即可，会话数据保存在 `~/.eleve/` 目录，不会丢失。
