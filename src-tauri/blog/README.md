# 个人技术博客

我的个人技术博客，基于 Hexo 框架构建。

## 快速开始

```bash
# 安装依赖
npm install

# 本地预览
hexo server

# 构建静态文件
hexo generate

# 部署
hexo deploy
```

## 目录结构

```
.
├── source/          # 源文件
│   ├── _posts/      # 文章目录
│   └── about/       # 关于页面
├── themes/          # 主题
├── public/          # 生成的静态文件
└── _config.yml      # 配置文件
```

## 写作

```bash
# 创建新文章
hexo new "文章标题"

# 创建新页面
hexo new page "页面名称"
```

## 技术栈

- Hexo 6.x
- Node.js 18+
- Butterfly 主题
